# Lumina Brand Voice Guide

> Every sentence on Lumina must leave the reader feeling slightly better than before they read it.

---

## The Four Voice Pillars

### WARM
Writes like a thoughtful friend, not a brand. Never formal. Never distant. Always present.

**IS:** "Your story matters here."
**IS NOT:** "We value your contribution to our platform."

Warmth means choosing the human word over the professional one. It means using "you" and meaning it. It never performs emotion — it simply is.

---

### HONEST
Says what it means. No spin, no inflation, no hyperbole. Earns trust by being true.

**IS:** "6 people have shared stories this week."
**IS NOT:** "Thousands of amazing stories await you!"

Honesty is the foundation of trust. We earn credibility by never overstating. If something is small, we say it's small. If something is difficult, we say so.

---

### UNHURRIED
Never rushes the reader. Breathes between thoughts. Respects the reader's pace.

**IS:** "Take your time. There's no rush here."
**IS NOT:** "Quick! Share your story today!"

Lumina exists in contrast to the attention economy. We never manufacture urgency. We never use countdown timers, "limited time" pressure, or FOMO language.

---

### ALIVE
Has personality and specificity. Avoids generic phrasing. Surprises gently.

**IS:** "Your words just landed somewhere new."
**IS NOT:** "Post published successfully."

Alive copy names things precisely. It replaces "content" with "story", "users" with "people", "submit" with "share". It notices what's specific about each moment and says that instead.

---

## Tone Dial

The same voice adjusts its tone based on context. One person, different moments.

| Context | Energy | Example |
|---------|--------|---------|
| **Celebratory** (story published, milestone) | High warmth, slight delight, brief | "Your story is out in the world. ✦" |
| **Welcoming** (onboarding, first visit) | Warm, curious, invitation-like, unhurried | "You found a quiet corner of the internet. Welcome." |
| **Informational** (settings, forms) | Clear, minimal, friendly but not chatty | "Choose what topics matter to you." |
| **Supportive** (errors, empty states) | Steady, non-judgmental, gently redirecting | "That didn't go through — your draft is safe." |
| **Poetic** (hero headings, daily prompts) | Literary, spacious, slightly lyrical | "The stories no one tells are the ones that change people." |
| **Serious** (community guidelines, safety) | Clear, direct, warm but unambiguous | "We take unkindness seriously here." |

## ALWAYS / NEVER Reference

| ALWAYS | NEVER |
|--------|-------|
| Use second person "you" | Use "users", "members", "folks" |
| Say "your story" | Say "content" or "posts" |
| Use contractions (it's, you've) | Sound formal or stiff |
| Speak in present tense | Use passive voice |
| Be specific over generic | Use vague superlatives (amazing, incredible) |
| Use short sentences alongside long | Write paragraphs without air |
| Name the emotion directly | Talk around feelings clinically |
| End CTAs with action verbs | End CTAs with "here" or "now" |
| Acknowledge difficulty honestly | Toxic positivity ("Everything is great!") |
| Use "and" at sentence start (OK) | Start with "We are pleased to..." |
| Write "3 stories" not "content" | Say "leverage", "synergy", "optimize" |
| Use em dashes for rhythm | Overuse exclamation marks |
| Let silence work (short copy wins) | Pad copy to fill space |
| Write "Share this story" | Write "Submit" |
| Write "Maybe later" | Write "Cancel" |
| Write "Something went wrong on our end" | Write "Error 500" |
| Write "A little longer makes it safer" | Write "Password must be 8 characters" |
| Write "That email doesn't look right" | Write "Invalid email address" |
| Write "Your draft is safe" | Write "Your changes were not saved" |
| Write "Take a breath?" | Write "Reading limit reached" |

## The 10-Point Copy Checklist

Before any copy ships, run it through these tests:

1. **WARM TEST** — Read it aloud. Does it sound like a thoughtful friend or a corporate brand? If corporate, rewrite.

2. **HONEST TEST** — Is any word inflated or exaggerated? "Amazing", "thousands", "incredible" — earn them or cut them.

3. **UNHURRIED TEST** — Does it pressure the reader? Any urgency that isn't real? Remove it. Lumina never rushes.

4. **SPECIFIC TEST** — Can you replace a generic word with a specific one? "stories" > "content". "say something" > "submit text".

5. **SHORT TEST** — Can any word be removed without changing the meaning? If yes, remove it. Every word must earn its place.

6. **SHAME-FREE TEST** — If it's an error message, does it blame the user? Rewrite to be collaborative, not accusatory.

7. **EMOTION TEST** — Does it acknowledge how the user might be feeling right now? Especially on sensitive pages.

8. **VOICE PILLAR TEST** — Check each: Warm ✓ Honest ✓ Unhurried ✓ Alive ✓. If one is missing, the copy isn't Lumina.

9. **CONTRAST TEST** — Read the copy next to a generic alternative. If they feel interchangeable, rewrite.

10. **LAST WORD TEST** — What's the last word a user reads before they act? That word carries the most weight. Make it count.
